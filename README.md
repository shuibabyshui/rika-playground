## English

### Overview
Rika Playground is a web-based sprite animation tool and playground designed for game developers and artists to quickly test and configure character animations. It allows you to upload sprite sheets, split them into frames, and preview them in a real-time game-like environment.

### Web Demo
[https://theamusing.github.io/Rika-Playground/](https://theamusing.github.io/Rika-Playground/)

### Key Features
- **Sprite Sheet Splitting**: Easily split uploaded sprite sheets by specifying columns and rows.
- **Frame Sequence Editing**: Mask unwanted frames to create clean animation sequences.
- **Real-time Preview**: Instantly see your animations in action with a controllable character.
- **Physics & Character Parameters**: Adjust walk/run speeds, jump power, gravity, and character scale.
- **Background Selection**: Choose from multiple backgrounds to test your character in different environments.
- **Keyboard Controls**: Full support for movement and special actions (WASD, Shift, J, H, 1-4).

### How to Use
1. **Select an Action**: Choose an animation state (e.g., Idle, Walk, Jump) from the left sidebar.
2. **Upload Sprite Sheet**: Click the upload area in the right sidebar to add your sprite sheet.
3. **Configure Splitting**: Set the number of columns and rows to correctly split the sheet into frames.
4. **Edit Frames**: In the bottom frame sequence, click the "X" on individual frames to disable them from the animation.
5. **Adjust Parameters**: Use the right sidebar to fine-tune physics and display settings.
6. **Play & Test**: Use the keyboard to control the character in the main preview area.

## 中文

### 项目简介
Rika Playground 是一个基于 Web 的序列帧动画工具和演练场，专为游戏开发者和美术人员设计，用于快速测试和配置角色动画。它支持上传精灵图（Sprite Sheet），将其切割成帧，并在实时游戏环境中进行预览。

### 在线演示
[https://theamusing.github.io/Rika-Playground/](https://theamusing.github.io/Rika-Playground/)

### 主要功能
- **精灵图切割**：通过指定行数和列数，轻松切割上传的精灵图。
- **序列帧编辑**：可以禁用（打叉）不需要的帧，从而创建干净的动画序列。
- **实时预览**：通过可操控的角色，立即查看动画效果。
- **物理与角色参数**：调整步行/跑步速度、跳跃力度、重力和角色缩放。
- **背景选择**：提供多种背景，以便在不同环境下测试角色。
- **键盘控制**：完整支持移动和特殊动作（WASD, Shift, J, H, 1-4）。

### 使用说明
1. **选择动作**：从左侧边栏选择一个动画状态（如 Idle, Walk, Jump）。
2. **上传精灵图**：点击右侧边栏的上传区域，添加您的精灵图。
3. **配置切割**：设置列数和行数，将图片正确切割为单帧。
4. **编辑帧序列**：在下方的序列帧列表中，点击单帧上的“X”将其从动画中禁用。
5. **调整参数**：使用右侧边栏微调物理参数和显示设置。
6. **测试与操控**：在主预览区域使用键盘控制角色进行测试。
